# DevFolio

개발자의 포트폴리오를 한 곳에서 모아보고 영감을 얻을 수 있는 **개발자 포트폴리오 큐레이션 플랫폼 프로젝트**입니다.  
본 프로젝트는 팀 프로젝트로 진행되었으며, 저는 개인 포트폴리오 페이지를 설계하고 구현했습니다.

---

# 1. 프로젝트 소개

### 서비스명
**DevFolio**

### 타겟 사용자
- 개발자 포트폴리오를 참고하고 싶은 취업 준비생
- 다양한 개발자 포트폴리오 UI를 탐색하고 싶은 개발자
- 신입 개발자의 역량을 빠르게 확인하고 싶은 채용 담당자

---

# 2. 기획 배경 & 문제 정의

### 기존 문제점

개발자 포트폴리오는 대부분 개인 블로그, GitHub, Notion 등  
여러 플랫폼에 흩어져 있어 한 곳에서 비교하거나 참고하기 어렵습니다.

또한 신입 개발자 입장에서는  
"다른 개발자들은 포트폴리오를 어떻게 만들었을까?"  
라는 궁금증을 해결하기 어렵다는 문제가 있습니다.

### 그래서 만들었습니다

DevFolio는 **개발자 포트폴리오를 큐레이션하여 한 곳에서 탐색할 수 있는 플랫폼**을 목표로 기획되었습니다.

사용자는 다양한 포트폴리오를 탐색하며 디자인과 구조를 참고할 수 있고,  
개발자는 자신의 포트폴리오를 소개할 수 있는 공간을 제공합니다.

---

# 3. 주요 기능 소개

### 1️. 개발자 포트폴리오 탐색

다양한 개발자의 포트폴리오를 확인할 수 있습니다.  
UI 구성과 프로젝트 정보를 참고할 수 있도록 설계되었습니다.

---

### 2️. 개인 포트폴리오 페이지

개발자의 기술 스택, 프로젝트 경험, 개발 기록 등을 정리한  
개인 포트폴리오 페이지를 제공합니다.

구성 예시

- About
- Skills
- Projects
- Dev Log
- Contact

---

### 3️. 반응형 UI

모바일 / 태블릿 / 데스크탑 환경에서도  
레이아웃이 자연스럽게 보이도록 반응형으로 설계했습니다.

---

### 4️. 인터랙티브 UI

사용자 경험을 향상시키기 위해 다음과 같은 인터랙션을 구현했습니다.

- 스크롤 기반 애니메이션
- 스크롤 진행바
- Canvas 기반 별 배경 인터랙션
- 메뉴 토글 인터랙션

# 6. 트러블슈팅 경험

### 문제 상황

JavaScript 인터랙션 기능이 정상적으로 동작하지 않는 문제가 발생했습니다.

브라우저 콘솔에서 다음과 같은 오류가 확인되었습니다.
'main.js' 파일 내부에 다음 코드가 포함되어 있었습니다. 
<script src="main.js"></script>
이 코드는 HTML 파일에서 사용되어야 하는 코드이기에 문법 오류가 발생하였습니다. 

### 해결 방법
- 'main.js'내부의 <script>태그 제거
- JavaScript 코드만 남기도록 수정
수정 후 인터랙티브 기능이 정상적으로 작동했습니다.
